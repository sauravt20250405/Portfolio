# Personal Portfolio

Welcome to my personal portfolio repository! This website showcases my journey as an AI & Data Science Engineer, highlighting my skills, education, experience, and the various projects I have worked on.

## 🚀 Live Demo
**[View My Portfolio Here](https://sauravt20250405.github.io/Portfolio/)**

## 🛠️ Technologies Used
This portfolio is built using modern web development practices without relying on heavy frameworks:
- **HTML5**: For semantic structure and accessibility.
- **CSS3**: For styling, responsive design (Flexbox/Grid), and advanced UI/UX elements like glassmorphism, gradients, and micro-animations.
- **Vanilla JavaScript**: For DOM manipulation, scroll animations (Intersection Observer API), typing effects, and dynamic number counters.
- **Web3Forms API**: For handling the functional contact form directly from the frontend.

## ✨ Key Features
- **Responsive Design**: Fully adaptable to mobile, tablet, and desktop screens.
- **Interactive UI**: Includes typing effects, scroll-reveal animations, dynamic number counting, and a custom mobile menu.
- **Project Showcase**: A dedicated section displaying 9 highlighted projects (ranging from AI platforms like Nexus AI OS, to web applications like a Translator App and Weather App).
- **Functional Contact Form**: Visitors can send messages directly to my email using the integrated Web3Forms setup.

## 📂 Project Structure
- `index.html` - The main HTML structure of the portfolio.
- `style.css` - Custom styles, layout designs, and CSS animations.
- `script.js` - JavaScript logic for interactivity and API integration.
