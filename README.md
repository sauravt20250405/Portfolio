## рџ› пёЏ Technologies & Tools Used

This portfolio is custom-built from the ground up using modern web technologies without relying on heavy frameworks like React or Angular. This ensures lightning-fast load times and maximum control over the DOM.

### Frontend
- **HTML5**: For semantic structuring and accessible web design.
- **CSS3**: For advanced styling, responsive design (CSS Grid and Flexbox), smooth transitions, glassmorphism UI effects, and modern gradients.
- **Vanilla JavaScript (ES6+)**: Handles all client-side logic, including:
  - Custom typing effect in the hero section.
  - Scroll-reveal animations using the `IntersectionObserver` API.
  - Dynamic, animated number counters for statistics.
  - Sticky navigation bar and smooth scrolling.
  - Interactive skill tabs.

### Backend / API Integrations
- **Web3Forms API**: Powers the "Contact Me" section, allowing visitors to send emails directly to me without needing a dedicated backend server.
